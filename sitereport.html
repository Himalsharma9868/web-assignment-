Learning Experience & Reflective Discussion

The module CSY1063 has been an experience in my life which has changed my life as I have acquired the fundamentals of web development. Being a beginner, I learned that coding does not only involve typing codes but also logic structure and imagination. My experience was marked with several highs and lows. The other major achievement was a successful application of a responsive CSS Grid on my project gallery; it was actually satisfying to see how the layout works so well with the different screen size. On the other hand, the biggest issue was in undergoing debugging process. The other thing that I wasted a lot of time on is troubleshooting of elements that did not match as anticipated due to the specificity and inheritance of CSS. These downs made me learn how to effectively use the browser developer tools to inspect items and resolve conflicts that is a valuable skill as a developer.

Design Decisions & Technical UI Development

Technically, the work on the development of this site was aimed at designing a high-contrast User Interface (UI) which is clean and reflects my professional identity. I selected a bright color scheme of "Black, White, and Red". The Call-to-Action buttons and headings are carefully placed with red color to direct the attention of the user, and the base of the site is black and white to create a professional appearance. In the case of typography, I chose the Segoe UI as it is a modern and a non-serif font that enhances readability.

As an element of my technical proficiency, I applied a complicated CSS Grid framework on the project.html page. In comparison to the linear home page, the project page has an asymmetrical grid to feature various project types, which demonstrates that I can manage advanced CSS positioning. I was impressed by the simplicity and blank space portfolios on websites such as Behance, which served as my major influence in this design. The site turned into more than simple wireframes into a multi-page and fully validated experience that is both pleasing to the eye and better performing than merely functional.
